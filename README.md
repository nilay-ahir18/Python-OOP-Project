# Python-OOP-Project
The Employee Management System is a menu-driven Python application developed using Object-Oriented Programming (OOP) concepts. It allows users to create and manage Person, Employee, and Manager records and display their details in a formatted manner.
This project demonstrates the practical use of classes, inheritance, method overriding, and loops in Python.

🎯 Objectives

To understand Object-Oriented Programming in Python

To implement inheritance using real-world examples

To create a menu-based console application

To display data in a clean and formatted output

🛠 Technologies Used

Language: Python 3
